## Executive Summary

The CMS RHC Enrollments file does **not** resolve to exactly 30 Washington rural clinic networks. The current CMS extract contains **110 WA RHC enrollment locations** grouped into **45 organization/network names**. Using the standard Eastern Washington county set, the Eastern WA subset contains **58 locations** across **28 networks**.

If the project target is specifically “30 networks,” the cleanest shortlist is the **top 30 statewide networks by RHC location count** below. If the target is Eastern Washington only, the CMS count is **28 networks / 58 locations**, not 30.

## Source and Method

- Pulled current CMS RHC Enrollments API data: `https://data.cms.gov/data-api/v1/dataset/3b7e7659-067e-41ea-8e36-f9ee2036e1f6/data?size=5000&page=1`.
- Checked CMS RHC Additional Addresses file: `https://data.cms.gov/sites/default/files/2026-04/RHC_Additional_Addresses_2026.04.01.csv`.
  - Result: **0 WA additional-address rows** in the CMS 2026-04-01 file, so WA multi-location data was represented through separate enrollment rows rather than additional-address rows.
- Checked CMS RHC All Owners file for owner-derived grouping: `https://data.cms.gov/sites/default/files/2026-01/7d234c00-e76a-4875-881c-afd0cd886207/RHC_All_Owners_2026.01.02.csv`.
  - Result: owner-derived grouping did **not** collapse the WA organization count below the enrollment organization count, so `network_name` uses the CMS `ORGANIZATION NAME`.
- Geocoded addresses with the Census Geocoder where possible.
  - Census matches: **103**.
  - Nominatim/city fallbacks: **7**.
  - One fallback is approximate: Republic Medical Clinic uses Republic city-center coordinates because the exact `10 ROS CIR` address did not geocode.

## Data Quality Notes

- CMS RHC data is current enrollment data from PECOS, not a historical panel.
- `network_name` is the CMS organization name, not a brand-normalized parent company.
- Duplicate-looking rows can represent distinct Medicare enrollments/NPIs or separate RHC practice locations.
- `lat`/`lon` are suitable for Mapbox plotting, but the Republic Medical Clinic coordinate should be treated as approximate until a better geocoder or Mapbox result is available.
